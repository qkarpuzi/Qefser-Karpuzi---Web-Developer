Project 1

A detailed look into one of my projects.

Books Review Website This is a user-friendly website designed for book enthusiasts to explore and review their favorite books. It features a clean, modern layout with a responsive design that works well on both desktop and mobile devices.

Key Features: Browse Books: View details of various books including cover images, titles, authors, genres, and ratings. Submit Reviews: Easily submit reviews for books, including title, author, rating, and a written review. Book Details: Get detailed information about books with interactive elements that showcase book cover images and detailed descriptions.


Tools Used: HTML/CSS: For structuring and styling the website. JavaScript: To handle form submissions and display book details dynamically. Google Fonts: For modern typography.

This project demonstrates my skills in web development, focusing on user interaction and responsive design.
